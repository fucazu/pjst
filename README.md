# pjst
Project to generate simple node structure with git ignore and npm init :rocket:

## Steps include
- `npm init -y`
- `git init`
- create `.gitignore`
- create `index.js`

## Install
```bash
$ npm i -g pjst
```

## Usage
Inside folder to create files:
```bash
$ pjst .
```
To create folder:
```bash
$ pjst folder_name
```
